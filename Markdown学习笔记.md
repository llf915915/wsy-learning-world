# Markdown学习笔记

（说明：此笔记由*[typora提供的语法学习网址][1]*、*小红书*查找、*B站*多个视频学习、向*豆包*寻求帮助而后实践检验，最后总结而成。）

（*注意：很多功能在不同编辑器里有不同简便编辑方式，本篇笔记只记录传统编辑方式，且由于在学习markdown时还不会使用此工具，最开始为纸质笔记，所以本篇笔记只做简写。*）

## 基础语法

### 块元素

1. **元数据块**：存储文档的基础信息。操作：输入---＋回车键;

2. **目录**：输入[[toc]][^footnote1]+回车键，有些编辑器不支持直接生成自动标题，需进行其他操作，具体见笔记[在第一阶段出现并解决的问题][2];

3. **标题**：由，#（可有1—6个，分别对应1—6级标题）+空格+文字标题内容，表示；

4. **引用文字**：由>+引用内容组成（用三个空格分隔两个块引用）:

5. **列表**：输入*/+/-加空格加内容可生成一个无序列表，输入数字加‘.’加内容可生成一个有序列表（这些内容之前的东西为列表的语法标记）;

6. **任务列表**：列表的语法标记+空格+[空格]+空格，便会自动生成小框，点击切换完成or未完成;

7. **表格**：|＋表头1+|+表头2+|+回车，用竖线分隔内容单元格，内容行按需增加，另外说明：

   | 对齐方式 | 左对齐    | 居中对齐     | 右对齐                 |
   | -------- | --------- | ------------ | ---------------------- |
   | 输入操作 | \|：---\| | \|：---：\|  | \|---：\|              |
   | 适用于   | 较长内容  | 带空格的文本 | 钱数等需靠右书写的内容 |

8. **脚注**：文本后加[ ^自定义标记名]+文章低端位置：[ ^自定义标记名]：注释内容；

9. **水平线**：---+回车/***+回车；

10. **代码块**：用一对三个反引号’ ’ ’圈住代码块，在反引号后面添加语言标识符；

11. **数学公式块**：$$后加公式表示代号。

### Span元素

1. **链接**：*内联式链接*：[ 文字]（网址）；*引用式链接*：正文：[ 文字] [1]+文档末尾：[1]：网址；
2. **内部链接**：*书签*：[会被显示的文字] （#标题名）；*参考样式链接*：正文：[链接显示的文字] [标签名]+文档任意位置：[标签名]：真实网址“提示词”（可不加提示词）；隐式快捷链接：
3. **URL**:用<>包裹网址；
4. **图片**：！[替代文字]（图片路径“提示文字”），操作方式：直接拖动（可能换个编辑器就无法显示了，需了解图床，见[在第一阶段出现并解决的问题][2]）；
5. **代码（短）**：一个反引号圈起来；
6. **删除线**：~~ 错误文字 ~~ ，显示出  ~~错误文字~~；
7. **下划线**：< u>内容< /u>，显示出<u>内容</u>；

*（还有一些特殊输入，在此不作说明）*

# HTML

## 简介

决定页面结构与内容（包括文本、图像、链接等）。

## HTML文件结构

1. < !DOCTYPE html > 证明是html文件（文档声明）

2. < html>< /html>开始和结束，包含整个文档

3. < head > < /head > 包含文件头部包括

   （1）< title>< /title>文件标题

   （2）< meta charset="UTF-8">文件编码格式

   （3）外部样式表（style）、CSS、JS

4. < body>< /body>包含实际显示在浏览器页面的内容，文本、图像、链接…（HTML内容填充）

*注意：在vscode中编写时只需要！+回车便会出现基本格式*



## HTML标签

### 分类

1. 单标签：<>
2. 双标签：<></\>

### 常用文本标签

1. 标题标签：<h1—h6></~>一级到六级标题（*快捷方式：输入h1—h6＋回车*）

2. 段落标签：< p>< /p>

3. 六件套：

   （1）加粗：< b>< /b> \ < strong>< /strong>

   （2）斜体：< i>< /i>

   （3）换行：< br>

   （4）水平线：< hr>

   （5）下划线：< u>< /u>

   （6）删除线：< s>< /s>

4. 列表标签：

   （1）无序列表：< ul>之间使用< li>元素< /li>

   （2）有序列表：< ol>之间使用~

5. 表格标签：

   * < table>（位于最外侧）

   * 列标题：< tr>包裹< th>
   * 表格内容：< tr>包裹< td>

6. 图片：< img src="定义要显示图片的路径 \ URL">

7. 其他：< a>< /a>常用于创建链接到其他的网页或者位置



## HTML属性

### 简介

用于描述元素的特征和行为，大致形式为：<开始标签 属性名=“属性值”>

### 常用文本标签

1. CSS中属性：

   （1）class：为HTML元素定义一个或多个类名

   （2）id：定义元素唯一的id

   （3）style：规定元素的行内样式

2. < a>相关属性：

   （1）href:定义了这个链接到的目标可以是其他网页的URL，文件的路径、email地址、手机号等

   （2）target：定义链接的打开方式，其属性值:

   1. _self:是个默认值，表示链接在当前窗口或标签页中打开
   2. _blank:表示链接将在新的窗口或标签页中打开
   3. _parent:表示链接将在副窗口或副框架中打开
   4. _top:表示在顶层窗口或顶层框架中打开

3. < img>相关属性：

   （1）src:定义图片文件的路径或URL

   （2）alt:定义文件的替代文本,图片加载不成功才显示的文字

   （3）width:图片宽度

   （4）height:图片高度

4. < table>相关属性：border="边框粗细"



## HTML元素

### 块元素

#### 简介

组织和布局页面，如段落、标题、列表、表格等，从新行开始，并占据整行的宽度，因此它们会在页面上呈现为一块独立的内容块

*注意：可以包含其他块级元素和行内元素*

#### 常见元素

< div>、< p>、< h1—h6>、< ul>、< ol>、< li>、< table>、< form>

##### 经典元素

div:仅用于组织内容，用于创建页面的布局结构，如页脚、页眉、导航栏、侧边栏、中间内容区域，e.g.导航栏（nav）：< div class="nav">< a href="#">内容< /a>< /div>

### 行内元素

#### 简介

在文本中插入小的元素，添加文本样式，通常只在同一行内呈现，不会独占一行，只占据其内容所需的宽度，而不是整行的宽度

*注意：不能包含块级元素，可包含其他行内元素*

#### 常见元素

< span>、< a>、< strong>、< em>、< img>、< br>、< imput>

##### 经典元素

span:把部分文本包括起来以对其使用样式（CSS、JS行为）

# CSS

## 简介

层叠样式表，用于定义网页样式和布局的样式表语言，指定页面中各个元素的颜色、字体、大小、间距、边框、背景等样式



## 格式

选择器{

属性：属性值;（英文的分号）

}



## CSS样式

### 分类

1. 内联样式：在CSS直接放在HTML元素中使用< style>标签，e.g.< h1 style="color="red">
2. 内部样式表：在< head>中用< style>
3. 外部样式表：CSS样式单独放到CSS文件中，在< head>中link到HTML文件中，满足重复使用，e.g.在HTML总文件中建一个CSS文件夹，在其中再建一个style.css，再在< head>中输入< link rel="style sheet" href="CSS文件路径">

### 优先级

内联样式>内部样式表>外部样式表

## CSS属性

### 常见属性

1. 字体：

（1）font在一个声明中设置所有字体相关属性

（2）字体颜色：color；背景颜色：background-color；字体大小：font-size；字体：font-family:'KaiTi(e.g.)'（中文拼音）；字体粗细：font-weight…（更多请见runoob菜鸟教程）

2. 盒子模型：

   （1）内容：content包含的内容、padding内容与边框距、margin外界与边框距；

   （2）实现：border属性：边框、样式 e.g.5px solid color（包括solid实线dashed虚线dotted点线double双实线）

   （*border-width遵循上右下左的顺序*）

3. 其他：

   *补充：行内块元素：水平方向上排列，可以设置宽度、高度、内外边距等块级元素的属性，可包含其他行内元素或块级元素*

   （1）设置块元素可以给background color定义width和height，行内元素span不行

   （2）行内块元素只设置width，自动宽度等于高度，再设置高度才会改变

   *总结：行内块元素和块元素可以设置宽高，行内块元素和行内元素默认宽度是文本宽度*

## 网页布局方式

### 分类

1. 标准流（普通流、文档流）网页按照元素的书写顺序依次排列
2. 浮动
3. 定位
4. 自适应布局：Flex、Grid

### 布局讲解

#### 浮动

##### 简介

可让多个快级元素在同一行显示，相对于父元素只在其内部移动

##### 格式

< style>{

float:(elf/right/none（浮动方位）)

}

##### 特性

1. 脱标
2. 一行显示，顶部对齐
3. 具备行内块元素

（*注意：若无height父元素border会坍塌，则需清除浮动，清除方式：在父元素里overflow:hidden（直接加）或添加一个 ·父class::after{}*）

#### 定位

##### 简介

可精准定位，但缺乏灵活性（值为负，则为反方向）

##### 定位方式

1. 相对定位：相对于元素在文档流中的正常位置进行定位

   命名：box-class

   在< style>内加position:relative;right/left/top/bottom:_px;

2. 绝对定位:相对于其他最近的1定位，负元素进行定位，不占据文档流（相对于父元素到哪儿，它就到哪儿）

   position:absolute;

3. 固定定位:相对于浏览器窗口进行定位，不占据文档流，固定在屏幕上的某个位置，不随滚动而移动

   position:fixed;

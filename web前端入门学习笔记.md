# HTML

## 简介

决定页面结构与内容（包括文本、图像、链接等）。

## HTML文件结构

1. < !DOCTYPE html > 证明是html文件（文档声明）

2. < html>< /html>开始和结束，包含整个文档

3. < head > < /head > 包含文件头部包括

   （1）< title>< /title>文件标题

   （2）< meta charset="UTF-8">文件编码格式

   （3）外部样式表（style）、CSS、JS

4. < body>< /body>包含实际显示在浏览器页面的内容，文本、图像、链接…（HTML内容填充）

*注意：在vscode中编写时只需要！+回车便会出现基本格式*



## HTML标签

### 分类

1. 单标签：<>
2. 双标签：<></\>

### 常用文本标签

1. 标题标签：<h1—h6></~>一级到六级标题（*快捷方式：输入h1—h6＋回车*）

2. 段落标签：< p>< /p>

3. 六件套：

   （1）加粗：< b>< /b> \ < strong>< /strong>

   （2）斜体：< i>< /i>

   （3）换行：< br>

   （4）水平线：< hr>

   （5）下划线：< u>< /u>

   （6）删除线：< s>< /s>

4. 列表标签：

   （1）无序列表：< ul>之间使用< li>元素< /li>

   （2）有序列表：< ol>之间使用~

5. 表格标签：

   * < table>（位于最外侧）

   * 列标题：< tr>包裹< th>
   * 表格内容：< tr>包裹< td>

6. 图片：< img src="定义要显示图片的路径 \ URL">

7. 其他：< a>< /a>常用于创建链接到其他的网页或者位置



## HTML属性

### 简介

用于描述元素的特征和行为，大致形式为：<开始标签 属性名=“属性值”>

### 常用文本标签

1. CSS中属性：

   （1）class：为HTML元素定义一个或多个类名

   （2）id：定义元素唯一的id

   （3）style：规定元素的行内样式

2. < a>相关属性：

   （1）href:定义了这个链接到的目标可以是其他网页的URL，文件的路径、email地址、手机号等

   （2）target：定义链接的打开方式，其属性值:

   1. _self:是个默认值，表示链接在当前窗口或标签页中打开
   2. _blank:表示链接将在新的窗口或标签页中打开
   3. _parent:表示链接将在副窗口或副框架中打开
   4. _top:表示在顶层窗口或顶层框架中打开

3. < img>相关属性：

   （1）src:定义图片文件的路径或URL

   （2）alt:定义文件的替代文本,图片加载不成功才显示的文字

   （3）width:图片宽度

   （4）height:图片高度

4. < table>相关属性：border="边框粗细"



## HTML元素

### 块元素

#### 简介

组织和布局页面，如段落、标题、列表、表格等，从新行开始，并占据整行的宽度，因此它们会在页面上呈现为一块独立的内容块

*注意：可以包含其他块级元素和行内元素*

#### 常见元素

< div>、< p>、< h1—h6>、< ul>、< ol>、< li>、< table>、< form>

##### 经典元素

div:仅用于组织内容，用于创建页面的布局结构，如页脚、页眉、导航栏、侧边栏、中间内容区域，e.g.导航栏（nav）：< div class="nav">< a href="#">内容< /a>< /div>

### 行内元素

#### 简介

在文本中插入小的元素，添加文本样式，通常只在同一行内呈现，不会独占一行，只占据其内容所需的宽度，而不是整行的宽度

*注意：不能包含块级元素，可包含其他行内元素*

#### 常见元素

< span>、< a>、< strong>、< em>、< img>、< br>、< imput>

##### 经典元素

span:把部分文本包括起来以对其使用样式（CSS、JS行为）
